# The Fleet

> 100+ AI vessels composing the Lucineer agent ecosystem. Every vessel is a git repo. The repo IS the agent.

## The Ecosystem

- **Layer 1 — Touch:** studylog.ai, activelog.ai, dmlog.ai, makerlog.ai, playerlog.ai, reallog.ai, businesslog.ai, dreamlog.ai, codelog.ai, coinlog.ai — "It just works."
- **Layer 2 — Operate:** cocapn.ai (runtime A2A/A2UI/A2C/MCP) / cocapn.com (membership/billing) — "Customize and manage my fleet."
- **Layer 3 — Build:** deckboss.ai (design) / deckboss.net (hardware) / capitaine.ai (education) / field-captain (technician CLI) — "Design systems, equip agents, open the hood."

## Lifecycle

Build (deckboss.ai) → Buy (deckboss.net) → Deploy (cocapn) → Operate (cocapn.ai) → Evolve (capitaine.ai) → Sustain (cocapn.com)

## Architecture

Every vessel: worker.ts (zero deps) + wrangler.toml + README.md
- Inline HTML, no build step
- CSP headers on every response
- /health endpoint always returns 200
- Fleet footer linking back to ecosystem
- BYOK v2 — your keys, your models, your data

## Flywheel

More installs → more profiles → more manufacturers → cheaper units → more installs. Trust (reputation scores) is the currency. People make livings as installers, servicers, product designers, micro-manufacturers.

## Key Concepts

- **Skills/Context/Equipment**: Equipment = pre-model input (sensors, cameras, APIs). Skills = how agent thinks. Context = runtime memory.
- **Character Sheet**: Auto-generated resource plan based on detected hardware.
- **Git-Agent Architecture**: The repo IS the agent. Fork, improve, share.
- **The Moat**: Free software + hardware convenience + trust network + open protocol.
- **Pay-for-convenience**: Free $0 / Standard $9 / Professional $29 / Enterprise $99/seat
